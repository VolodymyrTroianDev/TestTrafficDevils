# Vite Vue 3 Project Setup

This guide will help you set up a Vite Vue 3 project.

## Prerequisites

Make sure you have the following installed:
- Node.js (version 18+)
- npm

## Getting Started


1. **Install dependencies:**

    ```bash
    npm install
    ```

   

2. **Run the development server:**

    ```bash
    npm run dev
    ```


3. **Build for production:**

    ```bash
    npm run build
    ```
